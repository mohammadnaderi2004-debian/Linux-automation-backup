# Linux Automation & Backup Manager
A simple terminal-based tool to automate Linux tasks and manage backups.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/mohammadnaderi2004-debian/Linux-automation-backup.git
cd Linux-automation-backup
```

2. Create a virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python main.py
```

5. Follow the terminal menu to select actions:

```
1. Backup Folder  
2. List Processes  
3. Kill Process  
4. Run Command  
5. Services  
6. System Monitor  
7. Dashboard  
8. Exit
```
